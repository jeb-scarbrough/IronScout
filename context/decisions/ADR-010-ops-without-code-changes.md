# ADR-010: Operations Without Production Code Changes

## Status
Accepted

## Context
Hotfixes and manual DB edits increase operational risk.

## Decision
Routine operations must be possible without production code changes or direct DB edits.

## Alternatives Considered
- Manual hotfixes
- Fully automated remediation

## Consequences
Safer operations with upfront tooling cost.

## Notes
Governs admin tooling and runbooks.
